## Why Markdown Matters

- Our workflow: **Linear ‚Üí Dev ‚Üí PR ‚Üí Review ‚Üí Merge**
- Stories already descriptive ‚Üí PR descriptions optional
- Markdown + HTML improve:
  - PR review comments
  - GitHub Discussions
  - Documentation and READMEs

> Note: Good formatting reduces reviewer fatigue. ‚úÖ
